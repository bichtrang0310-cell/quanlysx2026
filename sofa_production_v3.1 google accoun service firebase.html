Kết nối Google Sheet
Chọn chế độ kết nối phù hợp với cách bạn thiết lập Google Sheet
Ưu điểm: Đơn giản, nhanh
Yêu cầu: Sheet phải chia sẻ công khai (Anyone with link)
Google Cloud Console → APIs & Services → Credentials → Create API Key
Ưu điểm: Bảo mật hơn, không cần chia sẻ công khai
Yêu cầu: Chỉ cần share Sheet cho email Service Account
Google Cloud Console → IAM & Admin → Service Accounts → Create
Service Account → Keys → Add Key → JSON → tải file về → dán nội dung vào đây
📋 Bước tiếp theo: Mở Google Sheet → Chia sẻ → Dán email bên dưới → Viewer
